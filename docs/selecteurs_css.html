Sélecteurs CSS - Guide Complet

Tableau exhaustif de tous les types de sélecteurs CSS avec exemples

Type Sélecteur Description Exemple
Sélecteur universel * Sélectionne tous les éléments
* {
  margin: 0;
  padding: 0;
}
Sélecteur d'élément element Sélectionne tous les éléments du type spécifié
p {
  color: blue;
}
Sélecteur de classe .class Sélectionne tous les éléments avec la classe spécifiée
.ma-classe {
  font-weight: bold;
}
Sélecteur d'ID #id Sélectionne l'élément avec l'ID spécifié
#mon-id {
  background-color: yellow;
}
Sélecteur d'attribut [attr] Sélectionne les éléments avec l'attribut spécifié
[target] {
  border: 1px solid red;
}
Sélecteur d'attribut (valeur exacte) [attr=valeur] Sélectionne les éléments avec l'attribut et la valeur exacte spécifiés
[type="text"] {
  width: 200px;
}
Sélecteur d'attribut (contient) [attr*=valeur] Sélectionne les éléments dont l'attribut contient la valeur spécifiée
[class*="btn"] {
  padding: 10px;
}
Sélecteur d'attribut (commence par) [attr^=valeur] Sélectionne les éléments dont l'attribut commence par la valeur spécifiée
[href^="https"] {
  color: green;
}
Sélecteur d'attribut (termine par) [attr$=valeur] Sélectionne les éléments dont l'attribut termine par la valeur spécifiée
[src$=".jpg"] {
  border: 2px solid #ccc;
}
Combinaison Sélecteur d'attribut d'une balise element[attr] Sélectionne l'attribut d'une balise
img[src] {
  border: 2px solid #ccc;
}
Sélecteur descendant A B Sélectionne tous les éléments B qui sont descendants de A
div p {
  color: red;
}
Sélecteur enfant direct A > B Sélectionne tous les éléments B qui sont des enfants directs de A
div > p {
  font-size: 1.2em;
}
Sélecteur adjacent A + B Sélectionne l'élément B qui suit immédiatement A
h1 + p {
  margin-top: 0;
}
Sélecteur de frères général A ~ B Sélectionne tous les éléments B qui suivent A (même niveau)
h1 ~ p {
  color: gray;
}
Pseudo-classe d'état :hover Sélectionne l'élément au survol de la souris
a:hover {
  text-decoration: underline;
}
Pseudo-classe d'état :focus Sélectionne l'élément qui a le focus
input:focus {
  border-color: blue;
}
Pseudo-classe structurelle :first-child Sélectionne le premier enfant de son parent
li:first-child {
  font-weight: bold;
}
Pseudo-classe structurelle :last-child Sélectionne le dernier enfant de son parent
li:last-child {
  border-bottom: none;
}
Pseudo-classe structurelle :nth-child(n) Sélectionne le n-ième enfant de son parent
tr:nth-child(odd) {
  background-color: #f2f2f2;
}
Pseudo-classe structurelle :not(selector) Sélectionne les éléments qui ne correspondent pas au sélecteur
div:not(.exclu) {
  display: block;
}
Pseudo-élément ::before Insère du contenu avant l'élément
p::before {
  content: "> ";
  color: red;
}
Pseudo-élément ::after Insère du contenu après l'élément
a::after {
  content: " (lien)";
  font-size: 0.8em;
}
Pseudo-élément ::first-letter Sélectionne la première lettre de l'élément
p::first-letter {
  font-size: 2em;
  font-weight: bold;
}
Pseudo-élément ::first-line Sélectionne la première ligne de l'élément
p::first-line {
  text-transform: uppercase;
}
Sélecteur multiple A, B Sélectionne tous les éléments A et B
h1, h2, h3 {
  font-family: Arial, sans-serif;
}
Sélecteur combiné A.class Sélectionne les éléments A avec la classe spécifiée
p.important {
  color: red;
}
Sélecteur combiné A#id Sélectionne l'élément A avec l'ID spécifié
div#header {
  background-color: #333;
}
