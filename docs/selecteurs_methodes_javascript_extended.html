Sélecteurs et Méthodes JavaScript

Guide complet pour manipuler le DOM - Version étendue

Méthode/Propriété Description Exemple
getElementById()
Sélecteur
Sélectionne un élément par son ID (retourne un seul élément)
Retourne: Element
const element = document.getElementById('monId');
getElementsByClassName()
Sélecteur
Sélectionne tous les éléments avec une classe spécifique
Retourne: HTMLCollection (live)
const elements = document.getElementsByClassName('maClasse');
getElementsByTagName()
Sélecteur
Sélectionne tous les éléments avec un nom de balise spécifique
Retourne: HTMLCollection (live)
const elements = document.getElementsByTagName('div');
getElementsByName()
Sélecteur
Sélectionne tous les éléments avec un attribut name spécifique
Retourne: NodeList (live)
const elements = document.getElementsByName('username');
querySelector()
Sélecteur
Sélectionne le premier élément correspondant au sélecteur CSS
Retourne: Element
const element = document.querySelector('.maClasse');
const element2 = document.querySelector('#monId div.maClasse');
querySelectorAll()
Sélecteur
Sélectionne tous les éléments correspondant au sélecteur CSS
Retourne: NodeList (static)
const elements = document.querySelectorAll('div.maClasse');
const elements2 = document.querySelectorAll('input[type="text"]');
closest()
Sélecteur
Remonte l'arbre DOM pour trouver l'ancêtre correspondant au sélecteur
Retourne: Element
const parent = element.closest('.container');
matches()
Sélecteur
Vérifie si l'élément correspond au sélecteur CSS
Retourne: Boolean
if (element.matches('.active')) {
  // Faire quelque chose
}
getAttribute()
Attribut
Récupère la valeur d'un attribut d'un élément
Retourne: String
const valeur = element.getAttribute('data-info');
const href = link.getAttribute('href');
setAttribute()
Attribut
Définit ou modifie la valeur d'un attribut
Retourne: void
element.setAttribute('data-info', 'nouvelle valeur');
element.setAttribute('disabled', 'true');
removeAttribute()
Attribut
Supprime un attribut d'un élément
Retourne: void
element.removeAttribute('data-info');
element.removeAttribute('disabled');
hasAttribute()
Attribut
Vérifie si un élément possède un attribut spécifique
Retourne: Boolean
if (element.hasAttribute('data-info')) {
  // Faire quelque chose
}
toggleAttribute()
Attribut
Ajoute un attribut s'il n'existe pas, le supprime sinon
Retourne: Boolean
element.toggleAttribute('hidden');
element.toggleAttribute('disabled', true); // Force l'activation
dataset
Attribut
Accède aux attributs data-* via un objet
Retourne: DOMStringMap
// Pour <div data-id="123" data-user-name="john">
const id = element.dataset.id; // "123"
const userName = element.dataset.userName; // "john"
classList.add()
Classe
Ajoute une ou plusieurs classes à un élément
Retourne: void
element.classList.add('active');
element.classList.add('class1', 'class2');
classList.remove()
Classe
Supprime une ou plusieurs classes d'un élément
Retourne: void
element.classList.remove('active');
element.classList.remove('class1', 'class2');
classList.toggle()
Classe
Ajoute une classe si elle n'existe pas, la supprime sinon
Retourne: Boolean
element.classList.toggle('active');
element.classList.toggle('active', true); // Force l'ajout
classList.contains()
Classe
Vérifie si un élément possède une classe spécifique
Retourne: Boolean
if (element.classList.contains('active')) {
  // Faire quelque chose
}
classList.replace()
Classe
Remplace une classe par une autre
Retourne: Boolean
element.classList.replace('old-class', 'new-class');
createElement()
Élément
Crée un nouvel élément HTML
Retourne: Element
const newElement = document.createElement('div');
const button = document.createElement('button');
createTextNode()
Élément
Crée un nœud de texte
Retourne: Text
const textNode = document.createTextNode('Hello World');
element.appendChild(textNode);
appendChild()
Élément
Ajoute un nœud enfant à la fin d'un élément parent
Retourne: Node
parentElement.appendChild(newElement);
removeChild()
Élément
Supprime un nœud enfant d'un élément parent
Retourne: Node
parentElement.removeChild(childElement);
insertBefore()
Élément
Insère un nœud avant un nœud de référence
Retourne: Node
parentElement.insertBefore(newElement, referenceElement);
replaceChild()
Élément
Remplace un nœud enfant par un autre
Retourne: Node
parentElement.replaceChild(newElement, oldElement);
cloneNode()
Élément
Clone un élément (true pour cloner en profondeur)
Retourne: Node
const clone = element.cloneNode(true); // Clone avec enfants
const cloneShallow = element.cloneNode(false); // Clone sans enfants
innerHTML
Élément
Obtient ou définit le contenu HTML d'un élément
Retourne: String
element.innerHTML = '<span>Nouveau contenu</span>';
const html = element.innerHTML;
textContent
Élément
Obtient ou définit le contenu textuel d'un élément
Retourne: String
element.textContent = 'Nouveau texte';
const text = element.textContent;
innerText
Élément
Obtient ou définit le texte visible d'un élément
Retourne: String
element.innerText = 'Texte visible';
const visibleText = element.innerText;
insertAdjacentHTML()
Élément
Insère du HTML à une position spécifique
Retourne: void
element.insertAdjacentHTML('beforebegin', '<div>Avant</div>');
element.insertAdjacentHTML('afterend', '<div>Après</div>');
insertAdjacentElement()
Élément
Insère un élément à une position spécifique
Retourne: Element
element.insertAdjacentElement('beforebegin', newElement);
insertAdjacentText()
Élément
Insère du texte à une position spécifique
Retourne: void
element.insertAdjacentText('beforebegin', 'Texte avant');
style.property
Style
Obtient ou définit une propriété de style inline
Retourne: String
element.style.color = 'red';
element.style.backgroundColor = '#fff';
element.style.fontSize = '16px';
style.property
Style
Obtient ou définit une propriété de style inline
Retourne: String
element.style.color = 'red';
element.style.backgroundColor = '#fff';
element.style.fontSize = '16px';
getComputedStyle()
Style
Récupère les styles calculés (incluant CSS)
Retourne: CSSStyleDeclaration
const styles = window.getComputedStyle(element);
const color = styles.getPropertyValue('color');
classList + CSS
Style
Applique des styles via l'ajout/suppression de classes
Retourne: void
// CSS: .active { color: red; font-weight: bold; }
element.classList.add('active');
element.classList.remove('inactive');
cssText
Style
Définit plusieurs styles en une seule fois
Retourne: String
element.style.cssText = 'color: red; font-size: 16px; background: white;';
setProperty()
Style
Définit une propriété CSS avec priorité
Retourne: void
element.style.setProperty('color', 'red', 'important');
removeProperty()
Style
Supprime une propriété de style
Retourne: String
element.style.removeProperty('color');
parentNode
Navigation
Récupère le nœud parent
Retourne: Node
const parent = element.parentNode;
parentElement
Navigation
Récupère l'élément parent
Retourne: Element
const parent = element.parentElement;
children
Navigation
Récupère les éléments enfants (exclut les nœuds texte)
Retourne: HTMLCollection
const children = element.children;
const firstChild = element.children[0];
childNodes
Navigation
Récupère tous les nœuds enfants (inclut texte, commentaires)
Retourne: NodeList
const childNodes = element.childNodes;
firstChild / lastChild
Navigation
Récupère le premier/dernier nœud enfant
Retourne: Node
const first = element.firstChild;
const last = element.lastChild;
firstElementChild / lastElementChild
Navigation
Récupère le premier/dernier élément enfant
Retourne: Element
const first = element.firstElementChild;
const last = element.lastElementChild;
nextSibling / previousSibling
Navigation
Récupère le nœud suivant/précédent au même niveau
Retourne: Node
const next = element.nextSibling;
const prev = element.previousSibling;
nextElementSibling / previousElementSibling
Navigation
Récupère l'élément suivant/précédent au même niveau
Retourne: Element
const next = element.nextElementSibling;
const prev = element.previousElementSibling;
contains()
Navigation
Vérifie si un élément contient un autre élément
Retourne: Boolean
if (container.contains(element)) {
  // L'élément est dans le container
}
forms[]
Formulaire
Accède aux formulaires de la page
Retourne: HTMLCollection
const firstForm = document.forms[0];
const myForm = document.forms['formName'];
elements[]
Formulaire
Accède aux éléments d'un formulaire
Retourne: HTMLFormControlsCollection
const input = form.elements['username'];
const email = form.elements.email;
value
Formulaire
Obtient ou définit la valeur d'un champ de formulaire
Retourne: String
const inputValue = input.value;
input.value = 'Nouvelle valeur';
checked
Formulaire
Obtient ou définit l'état d'une checkbox/radio
Retourne: Boolean
if (checkbox.checked) {
  // La checkbox est cochée
}
radio.checked = true;
selectedIndex
Formulaire
Obtient ou définit l'index de l'option sélectionnée
Retourne: Number
select.selectedIndex = 2;
const index = select.selectedIndex;
options[]
Formulaire
Accède aux options d'un élément select
Retourne: HTMLOptionsCollection
const selectedOption = select.options[select.selectedIndex];
submit()
Formulaire
Soumet un formulaire par programmation
Retourne: void
form.submit();
reset()
Formulaire
Réinitialise un formulaire
Retourne: void
form.reset();
focus()
Formulaire
Donne le focus à un élément
Retourne: void
input.focus();
blur()
Formulaire
Retire le focus d'un élément
Retourne: void
input.blur();
addOption() (custom)
Formulaire
Ajoute une option à un select (méthode personnalisée)
Retourne: void
const option = document.createElement('option');
option.value = 'val';
option.textContent = 'Texte';
select.appendChild(option);
addEventListener()
Événement
Attache un gestionnaire d'événement à un élément
Retourne: void
element.addEventListener('click', function(e) {
  // Code à exécuter au clic
});
element.addEventListener('click', handler, { once: true });
removeEventListener()
Événement
Supprime un gestionnaire d'événement d'un élément
Retourne: void
element.removeEventListener('click', handlerFunction);
dispatchEvent()
Événement
Déclenche un événement par programmation
Retourne: Boolean
const event = new Event('click');
element.dispatchEvent(event);
preventDefault()
Événement
Empêche le comportement par défaut d'un événement
Retourne: void
element.addEventListener('submit', function(e) {
  e.preventDefault();
  // Empêche l'envoi du formulaire
});
stopPropagation()
Événement
Empêche la propagation d'un événement dans le DOM
Retourne: void
element.addEventListener('click', function(e) {
  e.stopPropagation();
  // Empêche la propagation vers les parents
});
stopImmediatePropagation()
Événement
Empêche la propagation et l'exécution d'autres gestionnaires
Retourne: void
element.addEventListener('click', function(e) {
  e.stopImmediatePropagation();
  // Les autres gestionnaires click ne s'exécuteront pas
});
Event.target
Événement
Élément qui a déclenché l'événement
Retourne: Element
element.addEventListener('click', function(e) {
  const clickedElement = e.target;
});
Event.currentTarget
Événement
Élément sur lequel le gestionnaire est attaché
Retourne: Element
element.addEventListener('click', function(e) {
  const handlerElement = e.currentTarget;
});
once option
Événement
Le gestionnaire ne s'exécute qu'une seule fois
Retourne: void
element.addEventListener('click', handler, {
  once: true // S'exécute une seule fois
});
passive option
Événement
Améliore les performances pour les événements de scroll
Retourne: void
element.addEventListener('touchstart', handler, {
  passive: true // Améliore les performances
});
