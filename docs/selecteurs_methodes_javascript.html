Sélecteurs et Méthodes JavaScript

Guide complet pour manipuler le DOM

Méthode Description Exemple
getElementById()
Sélecteur
Sélectionne un élément par son ID
const element = document.getElementById('monId');
getElementsByClassName()
Sélecteur
Sélectionne tous les éléments avec une classe spécifique
const elements = document.getElementsByClassName('maClasse');
getElementsByTagName()
Sélecteur
Sélectionne tous les éléments avec un nom de balise spécifique
const elements = document.getElementsByTagName('div');
querySelector()
Sélecteur
Sélectionne le premier élément correspondant au sélecteur CSS
const element = document.querySelector('.maClasse');
querySelectorAll()
Sélecteur
Sélectionne tous les éléments correspondant au sélecteur CSS
const elements = document.querySelectorAll('div.maClasse');
getAttribute()
Attribut
Récupère la valeur d'un attribut d'un élément
const valeur = element.getAttribute('data-info');
setAttribute()
Attribut
Définit ou modifie la valeur d'un attribut
element.setAttribute('data-info', 'nouvelle valeur');
removeAttribute()
Attribut
Supprime un attribut d'un élément
element.removeAttribute('data-info');
hasAttribute()
Attribut
Vérifie si un élément possède un attribut spécifique
if (element.hasAttribute('data-info')) { // Faire quelque chose }
classList.add()
Classe
 Ajoute une ou plusieurs classes à un élément
element.classList.add('active');
classList.remove()
Classe
 Supprime une ou plusieurs classes d'un élément
element.classList.remove('active');
classList.toggle()
Classe
 Ajoute une classe si elle n'existe pas, la supprime sinon
element.classList.toggle('active');
classList.contains()
Classe
 Vérifie si un élément possède une classe spécifique
if (element.classList.contains('active')) { // Faire quelque chose }
createElement()
Élément
 Crée un nouvel élément HTML
const newElement = document.createElement('div');
appendChild()
Élément
 Ajoute un nœud enfant à la fin d'un élément parent
parentElement.appendChild(newElement);
removeChild()
Élément
 Supprime un nœud enfant d'un élément parent
parentElement.removeChild(childElement);
insertBefore()
Élément
 Insère un nœud avant un nœud de référence
parentElement.insertBefore(newElement, referenceElement);
innerHTML
Élément
 Obtient ou définit le contenu HTML d'un élément
element.innerHTML = '<span>Nouveau contenu</span>';
textContent
Élément
 Obtient ou définit le contenu textuel d'un élément
element.textContent = 'Nouveau texte';
addEventListener()
Événement
 Attache un gestionnaire d'événement à un élément
element.addEventListener('click', function() { // Code à exécuter au clic });
removeEventListener()
Événement
 Supprime un gestionnaire d'événement d'un élément
element.removeEventListener('click', handlerFunction);
preventDefault()
Événement
 Empêche le comportement par défaut d'un événement
element.addEventListener('submit', function(e) { e.preventDefault(); // Empêche l'envoi du formulaire });
stopPropagation()
Événement
 Empêche la propagation d'un événement dans le DOM
element.addEventListener('click', function(e) { e.stopPropagation(); // Empêche la propagation vers les parents });
